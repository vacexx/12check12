# 12check12
## Náležitosti
K sestavení projektu je potřeba:
* JUCE Framework: https://juce.com/get-juce/ (zahrnuje Projucer)
* C++ kompilátor/IDE: 
  * Windows: Visual Studio 2022 (https://visualstudio.microsoft.com/cs/downloads/).
 
## Jak sestavit projekt
1. Naklonujte si repozitář.
2. Sestavte JUCE projekt:
    * Otevřete Projucer.
    * Klikněte na File > Open a vyberte 12check12.jucer z repozitáře.
    * Klikněte na "Save Project and Open in IDE". Projucer následně automaticky sestaví JUCE projekt a otevře ho ve Visual Studio.
3. Sestavte řešení.
